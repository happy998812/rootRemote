# ⚡ RootRemote (Visual Shell Command Builder)
The world's first "block-assembly" style Visual Shell Command Builder designed specifically for rooted Android devices. It completely eliminates the typo stress of typing complex Linux commands on small mobile virtual keyboards.

### ✨ Key Features
- **One-Touch Command Building:** Easily assemble frequently used commands across 7 main categories (such as `ls`, `chmod`, `pkill`) with just a few clicks.
- **Modern Compose Architecture:** Powered by Jetpack Compose and Kotlin Coroutines for lightweight, asynchronous, and butter-smooth performance without draining your CPU.
- **High-Contrast Dark Theme:** Completely stripped away the default purple tint of Google Material 3, replacing it with a pitch-black background and neon green/orange high-contrast accents for maximum legibility.
- **Intelligent Flow Bypass:** System commands that require no extra arguments (like `reboot`) automatically skip optional steps, allowing direct execution in 0.5 seconds.
